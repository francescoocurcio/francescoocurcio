# Hello there! My name is Francesco 👋

I am a Computer Engineer holding a Master's degree in Artificial Intelligence & Machine Learning.

## 🧠 Focus on NLP & Large Language Models
My strongest technical interest lies in Natural Language Processing (NLP) and Large Language Models (LLMs). I continuously study the rapidly evolving AI landscape and actively experiment with the core technologies surrounding LLMs, including:

*   **Retrieval-Augmented Generation (RAG):** Integrating external knowledge bases with language models.
*   **Model Fine-Tuning:** Working with Parameter-Efficient Fine-Tuning (PEFT) techniques like LoRA.
*   **Agentic Workflows:** Building autonomous AI agents using frameworks like LangChain or LlamaIndex.
*   **Prompt Engineering:** Designing and optimizing complex prompts for state-of-the-art reasoning tasks.
*   **Vector Databases:** Managing and querying high-dimensional embeddings for semantic search.

I consistently monitor the latest research papers, state-of-the-art model releases, and industry news to stay updated on the most recent advancements in this ecosystem.

## 🚀 Approach to Learning
